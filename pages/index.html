Tybrode

Local-only macOS menu bar app. Works with allowlist apps only.

Tybrode records text diffs after accessibility-backed reads and generates local daily summaries. It does not upload capture content to any cloud service.

Download Latest DMG

Privacy first

  • Capture only when explicitly started.
  • Capture only for explicitly allowlisted apps.
  • No raw keystroke or clipboard bodies are stored.
  • Local Ollama summaries only.

Install

  1. Download Tybrode.dmg.
  2. Open it and move Tybrode.app to Applications.
  3. Launch from Applications and grant required macOS permissions.
